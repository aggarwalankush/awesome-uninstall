# MAC

```bash
# Android Studio
sudo rm -rf /Applications/Android\ Studio.app
sudo rm -rf ~/Library/Preferences/AndroidStudio*
sudo rm -rf ~/Library/Preferences/com.google.android.studio.plist
sudo rm -rf ~/Library/Application\ Support/AndroidStudio*
sudo rm -rf ~/Library/Logs/AndroidStudio*
sudo rm -rf ~/Library/Caches/AndroidStudio*
sudo rm -rf ~/.AndroidStudio*

# Gradle
sudo rm -rf ~/.gradle

# Android Virtual Devices(AVDs) and *.keystore. NOTE: If you using other IDE for Android Dev, don't delete this folder
sudo rm -rf ~/.android

# Android SDK tools
sudo rm -rf ~/Library/Android*

```

# UBUNTU

```bash
sudo rm -rf ~/.AndroidStudio
sudo rm -rf ~/.android
```
