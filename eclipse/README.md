# MAC

```bash
rm -rf ~/.eclipse 
rm -rf ~/.eclipse_keyring
rm -rf ~/Library/Saved\ Application\ State/org.eclipse.eclipse.savedState 
rm -rf ~/Library/Saved\ Application/org.eclipse.eclipse.savedState 
rm -rf ~/Library/Preferences/org.eclipse.eclipse.plist
rm -rf ~/Library/Caches/org.eclipse.platform.ide
rm -rf ~/Library/Caches/org.eclipse.eclipse
```

```bash
sudo apt-get purge eclipse*

```
